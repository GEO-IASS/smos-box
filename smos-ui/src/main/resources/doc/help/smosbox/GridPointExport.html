SMOS Grid Point Export

The Grid Point Export Tool can be used to write the data of selected DGG grid points into a CSV formatted text file. Individual values are separated by a semicolon.

Grid Point Export in VISAT

Source Product(s): Either the SMOS product selected in VISAT, or all SMOS products residing in a directory of the file system can be used for export. The search for SMOS products may optionally include subdirectories.

Region of Interest: There are three possibilities for specifying a region of interest (ROI) considered for export:

  • A ROI defined by a geometry
  • A ROI defined by the grid points under the pins of the SMOS product selected in VISAT
  • A ROI defined by a latitude-longitude box

Target File: The file, which receives the CSV output. The amount of data can be huge, depending on the ROI selected and the number of source products traversed.

Grid Point Export on the Command Line

The command line tool for exporting grid point data resides in the bin directory of your BEAM installation. From the console, type

    ${BEAM_HOME}/bin/export-grid-points.bat
    ${BEAM_HOME}/bin/export-grid-points.sh

or

    ${BEAM_HOME}/bin/export-grid-points.command

in order to obtain a usage message. The grid points, for which data is exported can be defined by either a latitude-longitude box or a latitude-longitude point. For instance, type

export-grid-points.sh -point 10.0 0.0 SM_TEST_MIR_SCLF1C_20070223T061043_20070223T061105_310_001_0

for dumping the data for a single grid point to the console. Type

export-grid-points.sh -box -10.0 -10.0 10.0 10.0 SM_TEST_MIR_SCLF1C_20070223T061043_20070223T061105_310_001_0

The file to which the data should be written can specified by means of the -o option. Otherwise stdout is used.

      export-grid-cells.sh -point 120 30 -o grid_cell_data.csv smosproduct.HDR

Multiple SMOS products from which data should be exported can be specified. The usual shell pattern can be used to specify multiple files:

      export-grid-cells.sh -point 120 30 -o grid_cell_data.csv repository/SM_*/SM*.HDR
