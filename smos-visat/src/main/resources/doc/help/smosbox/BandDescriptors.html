Band Descriptor Format

The DDDB band descriptor file is used to define which data of the SMOS Data Block Structures is displayed as a product band in BEAM. The band descriptor file is a line based text file (comma separated file) that is using the vertical-bar character ( |) as separator. A line beginning with a number-sign (# ) is treated as comment line.
