Introduction

The SMOS-Box is a collection of tools which aim at providing facilities for visualisation and basic analysis of SMOS Level 1 and 2 Data Products which fully complies with the BEAM Toolbox. Technical objective is the development of BEAM compliant software modules – all distributed under the GNU General Public License – for visualisation of measurement values, errors, flags and supplementary data contained in SMOS User Data Products (UDPs) as well as for advanced plotting and charting of data contained in the UDPs, and for exporting UDP data to alternative image and EO data formats.

This document provides the online help for Version 0.5 of the SMOS-Box, which is preliminary software released and distributed for the purpose of obtaining user feedback and additional user requirements at an early stage of development.

Usage

Version 0.5 of the SMOS-Box is comprised of four tools: the grid point brightness temperature data table, the grid point brightness temperature flag matrix diagram, the grid point brightness temperature chart, and the snapshot information tool. The tools can be invoked by clicking on corresponding icons arranged in the SMOS-Box toolbar.
